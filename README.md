## üîê Authentication & User Simulation

- **Login process simulation** ‚Äì mimics real authentication flow.

- **Form validation** ‚Äì checks if fields are filled correctly.

- **Fake database lookup** ‚Äì simulates checking whether a user exists in a database.

- **Redirection** ‚Äì users who are not logged in are automatically redirected to the login form page.
  <br>
  <br>
## Language Selection & Localization
- **Determining language based on user browser data**- After entering the website, the language in the browser is checked and automatically set

- **Dynamic language switching** ‚Äì users can select a preferred language.

- **Instant UI translation** ‚Äì interface text updates automatically after choosing a new language.

- **Language persistence** ‚Äì the selected language is stored in user preferences.
  <br>
  <br>
## Profile Management

- **Create profiles** ‚Äì users can create multiple profiles (just like Netflix).

- **Edit profile names** ‚Äì profile titles can be updated directly in the UI.

- **Delete profiles** ‚Äì users can remove unwanted profiles.

- **Active profile tracking** ‚Äì the system remembers which profile is currently selected.
  <br>
  <br>
## User Preferences & State Management

- **Persistent storage** ‚Äì saves user data (language, profiles, active profile, login status).

- **Session continuity** ‚Äì preferences are automatically loaded on page refresh.

- **Personalized experience** ‚Äì UI and data adapt based on the active profile and stored preferences.
  <br>
  <br>
## Navigation & Routing

- **Automatic redirects** ‚Äì ensures users always end up on the correct page depending on login state.

- **Logged-in users** ‚Üí redirected to browse page.

- **Not logged-in users** ‚Üí redirected to login form.
